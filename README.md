## тЬи Features (v1.0.0)

### ЁЯЦея╕П Terminal & Session Management

| Feature | Description |
|---------|-------------|
| **Multi-tab Interface** | Each session opens in its own dedicated full-size tab |
| **Split-screen Panes** | Split any tab horizontally or vertically via pane header buttons |
| **Tab Drag-to-Split** | Drag a tab to the left / right / top / bottom edge of the pane area тЖТ instant split view with resize handle |
| **Pane Drag-to-Swap** | Drag pane headers to reorder within a split layout |
| **Split View Splitter** | Drag the divider to resize split panes; тЬХ button to exit split mode |

### ЁЯУК Focus Info Bar

Real-time live stats for the currently **focused** server pane тАФ always visible at the top.

| Stat | Detail |
|------|--------|
| **Hostname** | Remote server hostname |
| **IP Address(es)** | Primary IP shown; `+N` badge for extras (hover for full list) |
| **CPU %** | Live usage via `/proc/stat` тАФ color coded green / yellow / red |
| **MEM %** | Used / Total MB via `free -m` тАФ color coded |
| **Uptime** | Human-readable via `uptime -p` |
| **Disk %** | Root partition usage тАФ click to open full `df -h` popup (container/virtual filesystems filtered) |

> Stats auto-refresh every **30 seconds** per focused pane.

### ЁЯЧВя╕П Session Sidebar

- Saved sessions with **quick connect**
- Preset OS command panels (Linux, Ubuntu, CentOS, Docker, etc.)
- **тШС All / тШР All** toggle тАФ select or deselect all panes in one click
- **Multi-Exec Bar** тАФ type one command, broadcast to all selected servers simultaneously

### ЁЯдЦ AI Assistant

- Built-in AI chat panel per pane (Claude / GPT-4o / Gemini / Groq)
- **Agent Mode** тАФ AI suggests command тЖТ executes in terminal тЖТ captures output тЖТ feeds back to AI (loop)
- Terminal output context bridge тАФ AI sees what you see
- All API keys encrypted with AES-256-GCM тАФ never leave your machine

### ЁЯУБ SFTP File Manager

- GUI file browser over SSH
- Upload / download files with progress
- Inline directory navigation

### ЁЯФР Security тАФ The Fortress

- **AES-256-GCM Vault** тАФ session credentials encrypted with master password (PBKDF2 key derivation)
- **Session Lock** тАФ lock the app; credentials wiped from memory until re-authenticated
- **Local-First** тАФ everything runs on `127.0.0.1:7654`; no cloud, no telemetry
- **Approval-Gate** тАФ AI-suggested commands require your confirmation before execution

### ЁЯкЯ Launcher

- **Windows** тАФ system tray app (`tray.ps1`), `launch.bat` for quick start
- **macOS / Linux** тАФ `launch.sh` / `stop.sh` shell scripts
- Auto-installs npm dependencies on first run

## тЪЩя╕П Installing Node.js

<details>
<summary><b>Windows</b></summary>

1. Go to **https://nodejs.org** and download the **LTS** version (`.msi` installer)
2. Run the installer тАФ click Next through all steps, keep all defaults
3. Open **Command Prompt** and verify:
   ```
   node -v
   ```
   You should see something like `v22.x.x`

> **Alternative:** Install via winget
> ```
> winget install OpenJS.NodeJS.LTS
> ```

</details>

<details>
<summary><b>macOS</b></summary>

**Option A тАФ Official installer (easiest)**
1. Go to **https://nodejs.org** and download the **LTS** `.pkg` file
2. Run the installer and follow the steps
3. Verify in Terminal:
   ```bash
   node -v
   ```

**Option B тАФ Homebrew (recommended for developers)**
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

```

</details>

</details>

## ЁЯЪА Quick Start

### Step 1 тАФ Download

```bash
git clone https://github.com/DEFiNE0223/DEFiNE-ZiON-AiTTY.git
cd DEFiNE-ZiON-AiTTY
```

> No git? Click **Code тЖТ Download ZIP** on GitHub, then extract.

---

### Step 2 тАФ Launch

**Windows** тАФ double-click `launch.bat`
> First run installs dependencies automatically. Check the system tray (bottom-right) for the icon.

**macOS / Linux**
```bash
chmod +x launch.sh stop.sh
./launch.sh
```

**Any OS (terminal)**
```bash
npm install   # first time only
node server.js
```

Then open **http://127.0.0.1:7654** in your browser.

---

### Step 3 тАФ First Setup

1. **Set a Master Password** тАФ encrypts all your data locally. Don't forget it.
2. **Add an SSH Session** тАФ click `+ Session` in the sidebar тЖТ enter host, port, username, password or private key.
3. **Connect** тАФ click your session. It opens as a full-size tab.

---

### Step 4 тАФ Working with Split View

- **H / V split**: click the split buttons in the pane header
- **Drag-to-split**: drag any tab and drop it on the **left / right / top / bottom** edge of the terminal area
- **Resize**: drag the splitter divider between panes
- **Exit split**: click the **тЬХ** button on the splitter

---

### Step 5 тАФ Enable AI Agent (Optional)

1. Click the **AI** tab in the left sidebar.
2. Click **Register** next to your preferred provider (Claude, GPT, Gemini, or Groq).
3. Paste your API key тАФ encrypted immediately, never leaves your machine.
4. Open any terminal pane тЖТ click the **ЁЯдЦ** button to open the AI chat.
5. Enable **Agent Mode** to let AI run commands and analyze results in a loop.

> **Where to get API keys:**
> - Claude тЖТ [console.anthropic.com](https://console.anthropic.com)
> - GPT тЖТ [platform.openai.com](https://platform.openai.com)
> - Gemini тЖТ [aistudio.google.com](https://aistudio.google.com)
> - Groq тЖТ [console.groq.com](https://console.groq.com)
